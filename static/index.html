              Flickr

              Flickr: go here to create a new flickr app. Select 'apply for a non-commercial key'. Name your app and describe it, hit submit and use the API key and API secret in the entry field above.


              Facebook

              Facebook: go here to create a new facebook app. Hit the 'create new app' button; name the app and hit 'continue'. Under 'select how your app integrates with facebook' set the site URL to the base URL for imageCatapult (so something like http://localhost:8080). Note the app ID and secret and use them in the entry field above.


              Tumblr

              Tumblr: go to here to register a new tumblr app. Give the app a name, and under 'default callback url' give it the base URL for imageCatapult and add '/auth/tumblrAuth' to the end, so something like: http://localhost:8080/auth/tumblrAuth. Create the app, make a note of the OAuth consumer key and secret and use them in the entry field above.


              Twitter

              Twitter go to here to create a new twitter app. Give the app a name and description and set the website and callback URL to the base URL for pyButer and add /auth/twitterAuth to the end, so something like: http://localhost:8080/auth/twitterAuth. Create the app, and make a note of the consumer key and secret and use them in the entry field above.
